# 🧠 Bori-AI: Módulo de Inteligencia Artificial Local

Este directorio contiene el motor de IA y los scripts de configuración del **Bori-System**, optimizados para ejecución en **Termux** y dispositivos móviles.

## 📍 Ubicación en el Proyecto
`Bori-System/modules/ai-bori`

## 🛠️ Especificaciones Técnicas
* **Motor:** `llama.cpp` (compilado para arquitectura ARM64/Android).
* **Modelo:** Qwen 2.5 0.5B Instruct (Formato GGUF).
* **Rol:** Mentor de Termux, experto en GitHub y analista de BoriCoin (Red Solana).

## 📋 Archivos Clave
* `install-ai.sh`: Script de automatización que descarga dependencias, compila el motor y baja el modelo.
* `bori-chat.sh`: El ejecutor principal que inicia la IA con el "System Prompt" personalizado de Bori-System.

## 🤝 Integración Bitrex & NFM
Este módulo está diseñado para servir como interfaz de lenguaje natural para el **Non Fungible Metaverse**. Permite a los usuarios consultar estrategias de inversión y comandos de terminal de forma offline, garantizando la privacidad de los datos en la red de Trujillo Alto.
